# App de Treino Premium

Arquivos para subir no GitHub de forma simples.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie estes arquivos:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. No repositório, vá em **Settings > Pages**.
4. Em **Build and deployment**, escolha:
   - **Source**: Deploy from a branch
   - **Branch**: `main` e pasta `/root`
5. Salve.
6. Aguarde alguns minutos. O link ficará parecido com:
   - `https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO/`

## Como usar no celular

1. Abra o link publicado no navegador do celular.
2. No Chrome Android, toque em **Adicionar à tela inicial**.
3. O site ficará com cara de app.
